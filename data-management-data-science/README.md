# Data Management for Data Science

Coursework focused on preparing messy data for analysis, using Python notebooks, querying databases, and organizing reproducible data science projects.

## Included Work

- `lab-01-log-file-processing/log-processing.ipynb` - parsing, cleaning, and summarizing messy log data.
- `lab-02-world-economy-analysis/world-economy-analysis.ipynb` - Pandas-based data cleaning, validation, and world economy analysis.
- `lab-03-amazon-reviews/amazon-reviews.ipynb` - product review cleaning and exploratory review analysis.
- `lab-04-databases/databases.ipynb` - DBMS concepts, JSON ingestion, SQLite tables, inserts, and SQL queries.
- `micro-skills-lab-01-python-basics/python-basics.ipynb` - Python basics practice notebook.
- `project-link.md` - link to the separate course project repository.

The course project stays in its own repository because it contains a complete data pipeline, reports, outputs, and reproducibility instructions.
