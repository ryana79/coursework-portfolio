# Data Management for Data Science

Coursework focused on preparing messy data for analysis, using Python notebooks, and organizing reproducible data science projects.

## Included Work

- `assignment-01/log-processing.ipynb` - notebook for parsing, cleaning, and summarizing messy log data.
- `assignment-01/approach-summary.txt` - short summary of the log-processing approach.
- `recitation-01/python-basics.ipynb` - Python basics practice notebook.
- `project-link.md` - link to the separate course project repository.

The course project stays in its own repository because it contains a complete data pipeline, reports, outputs, and reproducibility instructions.
