# Course Project: Stock Market Trend Analysis and Prediction

Repository: [ryana79/cs210-stock-project](https://github.com/ryana79/cs210-stock-project)

End-to-end data management and machine learning project for stock market trend analysis.

## Highlights

- SQLite database schema and SQL reporting queries.
- Data cleaning, validation, and exploratory analysis pipeline.
- Feature engineering and classification models for next-day price direction.
- Final report, metrics, generated figures, and contribution notes maintained in the standalone repository.
