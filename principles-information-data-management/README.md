# Principles of Information & Data Management

Coursework focused on relational databases, SQL, entity relationships, functional dependencies, and schema normalization.

## Included Work

- `homework-01/solution.html` - early database and information management exercises.
- `homework-03/queries.sql` - SQL schema creation and query practice.
- `homework-04/normalization.txt` - candidate keys, normal forms, and BCNF decomposition.
- `project-link.md` - link to the separate final project repository.

The final project remains in its own repository because it is a complete Java/MySQL application with its own setup instructions and source history.
