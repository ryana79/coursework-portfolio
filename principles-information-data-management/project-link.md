# Final Project: Online Travel Reservation System

Repository: [ryana79/online-travel-reservation-system](https://github.com/ryana79/online-travel-reservation-system)

Java Swing and MySQL travel reservation system built for the Principles of Information & Data Management final project.

## Highlights

- Customer, representative, and admin workflows.
- Flight search, reservations, cancellation, waiting list, and Q&A features.
- MySQL schema with primary keys, foreign keys, checks, indexes, and seed data.
- ER diagram and setup instructions maintained in the standalone repository.
