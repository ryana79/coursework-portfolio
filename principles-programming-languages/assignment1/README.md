## Assignment 1

Introductory OCaml exercises focused on recursion and list processing. The file `src/assignment1.ml` implements:
- `pow`, `range`, `flatten`, `remove_stutter`, `rotate`, `jump`, `nth`
- `digitsOfInt`, `additivePersistence`, `digitalRoot`

The module includes self-tests in `main`.

### Build
```bash
make
```

### Run tests
```bash
./assignment1
```
