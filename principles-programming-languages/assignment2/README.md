## Assignment 2

Higher-order functions and list combinators in OCaml. The file `src/assignment2.ml` implements:
- `cond_dup`, `n_times`, `zipwith`, `buckets`, `fib_tailrec`
- `sum_rows`, `ap`, `prefixes`, `powerset`, `assoc_list`

The module includes self-tests in `main`.

### Build
```bash
make
```

### Run tests
```bash
./assignment2
```
