# Design & Analysis of Computer Algorithms

Coursework from CS 344 focused on designing algorithms, proving correctness, and analyzing runtime.

## Included Work

- `homework-01/solution.tex` - asymptotic bounds, proofs, and recurrence-style reasoning.
- `homework-02/solution.tex` - selection, heap-based algorithms, greedy reasoning, and graph-related analysis.
- `homework-03/solution.tex` - dynamic programming and path reconstruction problems.

The assignments are kept in LaTeX to preserve the original mathematical formatting.
